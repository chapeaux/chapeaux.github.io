## The Chapeaux Playground

Chapellerie (French for "hat shop") is the Chapeaux playground — a space for demos, testing, and experimentation with web development tools from across the project.

## What's in the Playground?

The [Chapellerie repository](https://github.com/chapeaux/chapellerie) provides:

- **Component demos** — See Chapeaux tools in action with working examples
- **Integration tests** — Verify how components work together
- **Experimentation space** — Try out new ideas before they graduate to production

## Try It Yourself

Clone the repository and run it locally:

```bash
git clone https://github.com/chapeaux/chapellerie.git
cd chapellerie
```

Follow the setup instructions in the repository README to get the playground running on your machine.
